# 03 测试点分析

## 1. 前台用户模块

### 注册

- 注册页面是否正常打开
- 必填项为空是否提示
- 邮箱格式错误是否提示
- 重复邮箱是否限制注册
- 密码与确认密码不一致是否提示
- 注册成功后是否进入正确页面

### 登录

- 正确账号密码是否登录成功
- 错误密码是否提示
- 未注册邮箱是否提示
- 空账号或空密码是否提示
- 登录后用户菜单是否正确展示
- 退出登录后是否无法访问个人中心

## 2. 商品浏览模块

- 首页商品是否正常展示
- 商品分类是否可点击
- 商品详情页是否展示名称、价格、库存、图片、描述
- 商品搜索是否支持关键词查询
- 搜索无结果是否有明确提示
- 商品排序是否生效
- 商品上下架状态是否影响前台展示

## 3. 购物车模块

- 商品是否可以加入购物车
- 重复添加同一商品时数量是否累加
- 修改商品数量是否成功
- 删除购物车商品是否成功
- 数量为 0、负数、超库存时是否限制
- 购物车总价是否计算正确
- 未登录用户购物车行为是否符合预期

## 4. 下单结算模块

- 已登录用户是否可以进入结算页
- 未登录用户是否被引导登录或注册
- 地址必填项是否校验
- 配送方式是否可选择
- 支付方式是否可选择
- 提交订单后是否生成订单号
- 下单后后台是否可查询该订单

## 5. 后台登录与权限

- 后台登录页面是否正常打开
- 管理员账号密码正确是否登录成功
- 错误密码是否提示
- 退出登录后是否不能访问后台页面
- 普通用户是否不能访问后台管理功能

## 6. 后台商品管理

- 新增商品是否成功
- 商品名称为空是否限制保存
- 商品价格输入负数是否限制
- 商品库存输入异常值是否限制
- 编辑商品信息后前台是否同步变化
- 禁用商品后前台是否不展示
- 删除商品是否有二次确认

## 7. 后台订单管理

- 订单列表是否正常展示
- 是否支持按订单号、客户、状态筛选
- 订单详情是否展示商品、金额、收货信息
- 修改订单状态是否成功
- 订单状态流转是否合理
- 删除或取消订单是否有权限和确认限制

## 8. 客户管理

- 客户列表是否正常展示
- 是否支持按客户名称、邮箱筛选
- 客户详情是否展示完整
- 新增客户是否校验必填项
- 编辑客户信息是否保存成功
- 禁用客户后是否影响登录

## 9. 兼容性与易用性

- Chrome 浏览器核心流程是否正常
- Edge 浏览器核心流程是否正常
- 页面提示是否清晰
- 表单错误提示是否定位准确
- 按钮、链接、菜单是否易于理解
- 页面刷新或返回后数据是否保持合理

## 10. 安全与异常

- 未登录访问个人中心是否拦截
- 未登录访问后台是否拦截
- 表单是否限制明显非法输入
- 重复点击提交按钮是否产生重复订单
- URL 参数异常时页面是否报错
- 后台敏感操作是否需要管理员权限

## 11. Web 自动化测试补充

- 登录成功后是否能稳定进入用户中心
- 登录失败时是否能稳定识别错误提示
- 商品搜索后是否能断言目标商品出现在结果列表中
- 商品详情页加入购物车后，购物车数量是否发生变化
- 修改购物车商品数量后，总价是否重新计算
- 删除购物车商品后，购物车是否为空或商品行消失
- 下单主流程是否能执行到订单成功页
- 后台登录是否可作为后台自动化用例前置步骤
- 自动化执行失败时是否能保留截图、当前 URL 和错误信息

## 12. Postman 接口测试补充

- 登录接口是否能正确处理成功登录、错误密码、空账号等场景
- 商品查询接口是否返回商品名称、价格、库存等关键字段
- 商品搜索接口是否根据关键词过滤结果
- 购物车添加商品接口是否正确处理商品 ID 和数量
- 修改购物车数量接口是否限制 0、负数、超库存等异常值
- 删除购物车商品接口是否能正确更新购物车数据
- 创建订单接口是否返回订单号、金额和订单状态
- 查询订单接口是否能校验订单明细与前台提交信息一致
- 后台相关接口是否限制未登录或无权限访问

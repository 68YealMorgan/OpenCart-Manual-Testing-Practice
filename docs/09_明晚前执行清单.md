## 六、2026-06-06 晚上 23:00-24:00

### 1. Git 提交

```powershell

```

如果 Git 提示没有用户名和邮箱，先设置：

```powershell
git config --global user.name "你的英文名或GitHub用户名"
git config --global user.email "你的邮箱"
```

### 2. 推送 GitHub

先在 GitHub 新建公开仓库：

```text

```

然后执行：

```powershell

git remote add github https://github.com/你的用户名/OpenCart-Manual-Testing-Practice.git
git push -u github main
```

### 3. 推送 Gitee

先在 Gitee 新建公开仓库：

```text

```

然后执行：

```powershell

```
